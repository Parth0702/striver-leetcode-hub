# ⚡ CODEMATRIX - Futuristic Coding Hub

A dark, cyberpunk-themed coding platform that combines YouTube video learning, online code compilation, LeetCode practice, and AI analysis in one unified interface.

## 🌟 Features

### 🎬 **Embedded YouTube Player**
- **Watch videos directly** on the platform without redirects
- **Striver's DSA videos** and any programming content
- **Auto-embed** from YouTube URLs
- **Fullscreen support** with controls

### 💻 **Online Code Compiler**
- **Multi-language support**: Python, JavaScript, Bash
- **Real-time execution** with output display
- **Syntax highlighting** and error handling
- **Code persistence** during session

### 🎯 **LeetCode Integration**
- **Embedded LeetCode** practice interface
- **Problem search** and direct URL loading
- **Side-by-side** coding and problem viewing
- **Seamless navigation** between problems

### 🤖 **AI-Powered Analysis**
- **Video analysis**: Get AI summaries of YouTube content
- **Code explanation**: AI breaks down your algorithms
- **Learning insights**: Key concepts and recommendations
- **Real-time processing** with status updates

## 🎨 **Dark Futuristic Theme**

- **Cyberpunk aesthetics** with neon accents

## 🚀 **Live Demo**

**🌐 Deploy Options:**

### Option 1: GitHub Pages (Instant)
1. Go to [Settings → Pages](https://github.com/Parth0702/striver-leetcode-hub/settings/pages)
2. Source: "Deploy from branch" → "main"
3. **Live at**: `parth0702.github.io/striver-leetcode-hub`

### Option 2: Vercel (Full Features)
1. Visit [vercel.com/new](https://vercel.com/new)
2. Import your GitHub repo
3. Add environment variable: `BHINDI_API_KEY=your_key`
4. Deploy with full AI and compiler features

## 🎮 **How to Use**

### 🎬 **YouTube Panel**
```
1. Paste any YouTube URL (Striver, CodeWithHarry, etc.)
2. Click "🚀 Load Video"
3. Watch directly in the embedded player
4. Use AI analysis for video insights
```

### 💻 **Compiler Panel**
```
1. Select language (Python/JavaScript/Bash)
2. Write your code in the editor
3. Click "▶️ Run Code"
4. View output in real-time
```

### 🎯 **LeetCode Panel**
```
1. Search for problems or paste LeetCode URLs
2. Practice directly in the embedded frame
3. Switch between problems seamlessly
4. Use compiler for testing solutions
```

### 🤖 **AI Panel**
```
1. "🧠 Analyze Video" - Get AI insights from current video
2. "📝 Explain Code" - AI explains your algorithms
3. Real-time analysis with detailed breakdowns
4. Learning recommendations and next steps
```
